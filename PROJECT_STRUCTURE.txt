🚀 QUICK START
==============

1. Navigate to project:
   cd nexushub/backend

2. Install dependencies:
   npm install

3. Configure environment:
   cp .env.example .env
   # Edit .env with your settings

4. Start backend:
   npm run dev

5. Start frontend:
   cd ../frontend
   python -m http.server 3000

6. Open browser:
   http://localhost:3000
